Getting Started

Landy- is a HTML/CSS template based on Materialize framework. It allows to built impressive, better & faster, mobile friendly websites.

HTML Structure

This is the base of each HTML file:


                <!-- header - start
                ============================================= -->
                <header>
                ...
                </header>
                <!-- header - end
                ============================================= -->


                <!-- section - start
                ============================================= -->
                <section>
                ...
                </section>
                <!-- section - end
                ============================================= -->

                
                <!-- footer - start
                ============================================= -->
                <footer>
                ...
                </footer>
                <!-- footer - end
                ============================================= -->
